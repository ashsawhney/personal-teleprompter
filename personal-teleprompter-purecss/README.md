# Personal Teleprompter (Pure CSS)

Deployable Next.js teleprompter app for Vercel.

## Features

- pure CSS, no Tailwind dependency
- auto-scroll teleprompter
- voice-paced scrolling when browser speech recognition is available
- cue point jumping using lines like `[Intro]` or `[Cue: Closing]`
- fullscreen mode
- script save/load in local browser storage

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Upload all files and folders to GitHub, then import the repo into Vercel.
